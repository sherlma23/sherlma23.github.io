主要用于个人浏览器书签，用来分享有用的网站。


感觉不错的话，请帮我点一个 **Star**


本项目在 **所长导航** 基础上开发，详情参考 https://github.com/liutongxu/liutongxu.github.io

根据个人使用的需求，在Chatgpt的帮助下，我重写了部分代码以减少代码冗余，增加了实用的功能，比如：

1. 可以添加自定义网址，仅在本电脑使用，其他人看不到自定义网址。

2. 添加记事本功能，可以关联本地txt文件。在网页记事本修改的内容自动同步到本地 txt 文件中，在本地 txt 文件修改的内容自动同步到网页显示。配合文件同步的软件，可以实现多端待办事项同步。
